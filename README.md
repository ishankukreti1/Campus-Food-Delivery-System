# 🍔 Campus Food Delivery System (DAA Project)

## 📌 Overview
This project is a simulation of a campus food delivery system built using C++. It demonstrates the application of Design and Analysis of Algorithms (DAA) in a real-world scenario.

Users can place food orders, and the system optimizes delivery using multiple algorithms.

---

## ⚙️ Algorithms Used

- **Merge Sort** → Groups orders based on delivery location  
- **Job Sequencing** → Assigns delivery boys efficiently  
- **Knapsack Algorithm** → Optimizes delivery capacity  
- **Dijkstra’s Algorithm** → Finds shortest delivery path  
- **Aging Technique** → Prevents starvation of low-priority orders  

---

## 🧠 Features

- Multiple item ordering system  
- Priority-based order processing (price + urgency)  
- Smart delivery assignment (location + capacity based)  
- Prevention of order starvation using aging  
- Modular code using header and source files
